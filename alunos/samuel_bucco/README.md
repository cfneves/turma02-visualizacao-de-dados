### Conceitos aprendidos

* Clonagem de repositórios do GitHub
* Uso básico do terminal
* Configuração do VS Code
* Instalação de bibliotecas Python
* Introdução aos arquivos `.ipynb`
* Conceitos básicos de Git

### Comandos utilizados

#### Clonar repositório

```bash
git clone URL_DO_REPOSITORIO
```

#### Verificar repositórios remotos

```bash
git remote -v
```

#### Atualizar repositório local

```bash
git pull --rebase
```

#### Enviar alterações para o GitHub

```bash
git push
```

#### Abrir projeto no VS Code

```bash
code .
```

#### Instalar bibliotecas Python

```bash
python3 -m pip install pandas
python3 -m pip install openpyxl
python3 -m pip install notebook jupyter
```

### Bibliotecas instaladas

* pandas
* openpyxl
* jupyter
* notebook

### Observações

* O comando `code` precisa ser habilitado no VS Code através da opção:

```text
Shell Command: Install 'code' command in PATH
```

* Arquivos `.ipynb` são notebooks interativos utilizados em Data Science.
