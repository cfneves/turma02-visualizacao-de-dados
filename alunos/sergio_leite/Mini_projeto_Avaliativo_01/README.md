Mini-Projeto Avaliativo: Modelagem e Limpeza de Dados
**Curso:** Análise de Dados
**Estudante:** Sérgio Leite

Este projeto consiste no diagnóstico, limpeza, tratamento e extração de estatísticas descritivas de uma base de dados real de vendas e clientes. O principal objetivo foi aplicar as melhores práticas de manipulação de dados utilizando a biblioteca Pandas em Python para garantir a integridade das informações para futuras análises de negócio.

Tecnologias e Bibliotecas Utilizadas
**Python 3**
**Pandas:** Para toda a manipulação, limpeza e agregação dos dados.
**Jupyter Notebook / VS Code:** Ambiente de desenvolvimento.

Como Executar o Projeto
1. Certifique-se de ter o Python e o Pandas instalados na sua máquina.
2. Baixe o arquivo do notebook `.ipynb` e a base de dados na mesma pasta.
3. Abra o notebook no VS Code ou Jupyter e execute as células em ordem cronológica.
