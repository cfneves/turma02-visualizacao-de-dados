| Função                 | Para que serve                                 |
| ------------------------ | ---------------------------------------------- |
| `pd.read_excel()`      | Lê um arquivo Excel e transforma em DataFrame |
| `df.head(n)`           | Mostra as n primeiras linhas                   |
| `df.info()`            | Mostra tipos de dados e contagem de nulos      |
| `df.describe()`        | Estatísticas básicas das colunas numéricas  |
| `df.isnull().sum()`    | Conta os nulos por coluna                      |
| `df.fillna(valor)`     | Preenche nulos com um valor definido           |
| `df.dropna()`          | Remove linhas com nulos                        |
| `df.duplicated()`      | Identifica linhas duplicadas                   |
| `df.drop_duplicates()` | Remove as duplicatas                           |
| `str.strip()`          | Remove espaços no início/fim do texto        |
| `str.title()`          | Padroniza capitalização                      |
| `replace(dicionário)` | Substitui valores específicos                 |
| `pd.to_numeric()`      | Converte coluna para número                   |
| `np.where()`           | Aplica condição: se verdadeiro X, senão Y   |
| `df.to_csv()`          | Salva o DataFrame como arquivo CSV             |
| `df.to_excel()`        | Salva o DataFrame como arquivo Excel           |
