# Mini-Projeto — Análise Exploratória de Dados (Base Varejo)

*Aluno* Waldinei Lameira
*Curso:* Analista de Dados — Visualização de Dados
*Dataset:* [Base Varejo — Kaggle](https://www.kaggle.com/datasets/namespaiva/base-varejo)

## O que foi feito

Análise exploratória completa sobre uma base real de varejo brasileiro com 830 mil registros, passando pelas etapas de importação, transformação, limpeza, estatística descritiva, agrupamentos, geração de insights e visualização de dados.

## Principais resultados

- Base limpa: **733.447 registros** (96.553 duplicatas removidas — 11,6%)
- Categoria mais vendida: **ALIMENTOS** (52,4% das transações)
- Perfil de compras: **52,1% feminino** e 47,9% masculino
- Segmento predominante: **B** com 63,9% das compras

## Tecnologias

`pandas` · `kagglehub` · `matplotlib` · `pathlib`
