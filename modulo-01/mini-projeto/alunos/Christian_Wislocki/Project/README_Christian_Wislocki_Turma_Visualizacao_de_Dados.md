## 📌 Pré-requisitos de Ambiente

Para que o script seja executado com sucesso, certifique-se de que o **Python** esteja instalado em sua máquina, juntamente com as bibliotecas essenciais de análise de dados.

Caso não tenha as bibliotecas instaladas, execute o comando abaixo no seu terminal antes de rodar o projeto:
```bash
pip install pandas numpy matplotlib seaborn

## ▶️ Como Executar o Código

A execução do projeto foi simplificada para rodar inteiramente via linha de comando. Siga os passos abaixo:

1. Abra o seu **Terminal** (ou o terminal integrado do VSCode / Prompt de Comando).
2. Navegue até a pasta onde os arquivos do projeto foram extraídos.
3. Digite o comando abaixo e pressione `Enter`:

```bash
python Projeto_Varejo.py
