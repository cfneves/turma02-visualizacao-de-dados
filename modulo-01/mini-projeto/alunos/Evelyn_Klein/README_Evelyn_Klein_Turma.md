# Instrucoes de Execucao

Abra o arquivo `Mini_Projeto.ipynb` no VSCode ou no Google Colab e execute todas as celulas em ordem.

Se preferir, voce tambem pode rodar a versao em script:

```bash
python mini_projeto_avaliativo_modulo_1_semana_07.py
```
