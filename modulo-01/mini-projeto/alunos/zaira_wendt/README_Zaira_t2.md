## Como Executar o Projeto

### Pré-requisito

O arquivo **Base Varejo.csv** não está disponível neste repositório devido ao seu tamanho.

Antes da execução, salve o arquivo Base Varejo.csv na mesma pasta do projeto.

### Execução no VS Code

1. Abra a pasta do projeto no VS Code.
2. Certifique-se de que o arquivo Base Varejo.csv está na mesma pasta do script.
3. Abra o terminal.
4. Execute:

python miniprojeto_varejo.py

### Execução no Jupyter Notebook

1. Abra o arquivo projeto_varejo.ipynb.
2. Certifique-se de que o arquivo Base Varejo.csv está na mesma pasta do notebook.
3. Execute todas as células em ordem.
