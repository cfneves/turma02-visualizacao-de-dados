# Mini-projeto de visualização de dados [QA VDBI]

# Instruções para utilizar:
    • Abra a pasta mini-projeto.
    • Entre no projeto.ipynb.
    • Rode todas as celulas do notebook de cima para baixo.
    • Realize as leituras geradas após rodar.

# Aviso:
    Você precisa instalar as bibliotecas para poder visualizar as análises feitas.
    Ex:
        Foram criados gráficos para visualizar os dados:
        • Número de Compras por Ano.
        • Número de compras por Gênero.
        • Visualização da Distribuição de Categoria. 
    * após rodas as celulas eles apareceram para você poder visualizar sem as bibliotecas vai dar erro.

# Funcionalidade:
    • Tratamento da coluna DATA para DataTime.
    • Limpeza de colunas (Unnamed).
    • Criação de colunas de datas, (ano, mês, dia, semana, mês_ano).
    • Tratamento da Categoria "#N/D" na coluna PR_CAT 
        (Feita a troca de "#N/D" para "PRODUTO_SEM_CATEGORIA").
    • Tratamento de um erro cometido junta (sem categoria com PRODUTO_SEM_CATEGORIA).
    • Agrupamentos para visualização de dados (Número de Compras por Ano, Número de compras por Gênero).


# Oque é esperado:
    • Gráfico da distribuição de categorias.
    • Gráfico da evolução de compras por ano.
    • Gráfico número de compras por Gênero.
    • Dados tratados sem valores nulos ou categorias inválidas.
