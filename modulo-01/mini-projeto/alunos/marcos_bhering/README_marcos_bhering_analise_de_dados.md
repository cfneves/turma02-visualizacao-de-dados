# Mini-projeto de visualização de dados [T2]

# Descrição:
    este projeto tem como finalidade de aplicar conceitos de limpeza de dados, tratamento com python e visualização de dados  panda e matplotlib.

    Forão criados gráficos para visualizar os dados:
     • Número de Compras por Ano
     • Número de compras por Gênero
     • Visualização da Distribuição de Categoria 

# Funcionalidade:
    • Tratamento da coluna DATA para DataTime.
    • Limpeza de colunas (Unnamed).
    • Criação de colunas de datas, (ano, mês, dia, semana, mês_ano).
    • Tratamento da Categoria "#N/D" na coluna PR_CAT 
        (Feita a troca de "#N/D" para "PRODUTO_SEM_CATEGORIA").
    • Tratamento de um erro cometido junta (sem categoria com PRODUTO_SEM_CATEGORIA).
    • Agrupamentos para visualização de dados (aNúmero de Compras por Ano, Número de compras por Gênero).

# Tecnologias usadas:
    • Python
    • Matplotlib
    • Pandas
